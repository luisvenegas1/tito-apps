# Assets de Tito Apps

Estos SVG son **PROVISIONALES**, generados con los colores oficiales para
poder trabajar. Reemplazalos por los archivos oficiales aprobados cuando los
tengas (mismos nombres para no cambiar rutas):

- `logo-titoapps.svg` — logotipo horizontal (verde + azul oscuro).
- `icon-titoapps.svg` — ícono/app icon.
- `favicon.svg` — favicon.

## Reglas

- La **mascota del billete es exclusiva de SplitPay**. No se usa como logo de
  Tito Apps ni se copia a GolPay o MoneyTrack.
- Cada app mantiene sus propios íconos de producto en su carpeta `public/`
  (ej. `apps/golpay/public/icon-512.png`). Estos assets son de la **marca madre**.
- Iconografía de interfaz (dentro de la UI): se usa `lucide-react`, no imágenes.
