# brand/icon — Isotipo e íconos de app

Isotipo "t·" para usos como ícono. Archivos reales principales:

| Archivo | Uso |
|---------|-----|
| `icon.svg` · `isotipo.svg` | Isotipo vectorial. |
| `icon-white.svg` · `icon-black.svg` | Isotipo en blanco / negro. |
| `icon.png` · `isotipo.png` | Isotipo PNG (1024). |
| `icon-light.png` · `icon-dark.png` | Tesela clara / oscura. |
| `app-icon-1024.png` · `app-icon-512.png` · `app-icon-192.png` | App icons. |
| `apple-touch-icon.png` | 180×180 iOS. |
| `favicon.svg` · `favicon.ico` · `favicon.png` · `favicon-16/32/48.png` | Favicons. |

Los íconos por-producto (ej. GolPay ⚽) viven en el `public/` de cada app, no aquí.

> **Limpieza pendiente (hacer en tu Mac):** esta carpeta contiene además copias de
> `logo-horizontal.*`, `logo-vertical.*`, `logo-white.svg` y `logo-dark.svg` que
> pertenecen a `logo/`, y favicons que también están en `favicon/`. Conviene
> borrar los duplicados para dejar `icon/` solo con el isotipo/íconos.
